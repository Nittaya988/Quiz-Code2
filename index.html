สอนวิธีอัปโหลดแบบละเอียด

ฉันจะสอนวิธีอัปโหลดแบบละเอียดทีละขั้นตอนครับ! เริ่มจากวิธีที่ง่ายที่สุดก่อน

🌟
วิธีที่ 1: Netlify (ง่ายที่สุด - แนะนำสำหรับมือใหม่)

ขั้นตอนที่ 1: เตรียมไฟล์

  1. เปิด Notepad หรือ Text Editor
  2. คัดลอกโค้ดแบบทดสอบทั้งหมด
    (ที่เราทำไว้)
  3. บันทึกเป็นไฟล์
    ชื่อ
    index.html
    • เลือก "Save as type" → "All Files"
    • ตั้งชื่อ:
      index.html
    • บันทึกไว้ที่ Desktop

ขั้นตอนที่ 2: สมัคร Netlify

  1. ไปที่
    https://netlify.com
  2. คลิก "Sign up"
  3. เลือกสมัครด้วย Google
    (ง่ายที่สุด)
  4. ยืนยันอีเมล
    (ถ้าจำเป็น)

ขั้นตอนที่ 3: อัปโหลดไฟล์

  1. หลังจากเข้าสู่ระบบ
    จะเห็นหน้า Dashboard
  2. มองหาส่วน "Want to deploy a new site without connecting to Git?"
  3. คลิก "Browse to upload"
    หรือ
    ลากไฟล์
    index.html
    วางลงในกล่อง
  4. รอสักครู่
    (ประมาณ 10-30 วินาที)

ขั้นตอนที่ 4: รับ URL

  1. จะได้ URL แบบ:
    https://wonderful-name-123456.netlify.app
  2. คลิกที่ URL เพื่อทดสอบ
  3. เสร็จแล้ว!
    🎉

🔧
วิธีที่ 2: GitHub Pages (เหมาะสำหรับใช้งานระยะยาว)

ขั้นตอนที่ 1: สมัคร GitHub

  1. ไปที่
    https://github.com
  2. คลิก "Sign up"
  3. กรอกข้อมูล:
    • Username:
      your-name-quiz
      (ตัวอย่าง)
    • Email: อีเมลของคุณ
    • Password: รหัสผ่าน
  4. ยืนยันอีเมล

ขั้นตอนที่ 2: สร้าง Repository

  1. คลิก "New repository"
    (ปุ่มสีเขียว)
  2. ตั้งชื่อ Repository:
    flood-quiz
    (หรือชื่ือที่ต้องการ)
  3. เลือก "Public"
  4. เช็ค "Add a README file"
  5. คลิก "Create repository"

ขั้นตอนที่ 3: อัปโหลดไฟล์

  1. คลิก "uploading an existing file"
  2. ลากไฟล์
    index.html
    วางลงในกล่อง
  3. เลื่อนลงล่าง กรอก:
    • Commit message:
      Add quiz file
  4. คลิก "Commit changes"

ขั้นตอนที่ 4: เปิดใช้ GitHub Pages

  1. คลิกแท็บ "Settings"
    (ด้านบน)
  2. เลื่อนลงหา "Pages"
    (เมนูซ้าย)
  3. ในส่วน "Source" เลือก:
    • Source: "Deploy from a branch"
    • Branch: "main"
    • Folder: "/ (root)"
  4. คลิก "Save"
  5. รอ 2-5 นาที
    จะได้ URL:
    https://username.github.io/flood-quiz

📱
วิธีที่ 3: Google Sites (ง่าย แต่มีข้อจำกัด)

ขั้นตอนที่ 1: เข้า Google Sites

  1. ไปที่
    https://sites.google.com
  2. เข้าสู่ระบบด้วย Google Account

ขั้นตอนที่ 2: สร้างไซต์

  1. คลิก "สร้าง"
    (หรือ "Create")
  2. เลือก "เว็บไซต์เปล่า"
  3. ตั้งชื่อไซต์:
    "แบบทดสอบ Code 2"

ขั้นตอนที่ 3: เพิ่มโค้ด

  1. คลิก "แทรก" → "Embed"
  2. เลือก "Embed code"
  3. วางโค้ดแบบทดสอบทั้งหมด
  4. คลิก "แทรก"

ขั้นตอนที่ 4: เผยแพร่

  1. คลิก "เผยแพร่"
    (มุมขวาบน)
  2. ตั้งชื่อ URL:
    quiz-code2
    (หรือชื่อที่ต้องการ)
  3. คลิก "เผยแพร่"
  4. จะได้ URL:
    https://sites.google.com/view/quiz-code2
